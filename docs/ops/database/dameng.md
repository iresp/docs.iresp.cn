# 武汉达梦数据库

[武汉达梦数据库](https://www.dameng.com/)

## 安装注意事项

::: tip 注意
初始化数据库实例时，编码需要改为```UTF-8```，去掉```字符串大小写敏感```选项。一旦数据库实例被创建参数将不可修改。
:::
