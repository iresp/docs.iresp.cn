# Aliyun ECS

此设备为云服务器，具备互联网访问能力、7x24小时稳定在线能力，可部署对外服务、生产环境。

  • IP: 47.95.66.119
  • Domain: ecs.iresp.cn

# 登录服务器

此服务器禁止使用密码登录！请先生成 SSH Key 将公钥发送给管理员。

ssh -p 2222 root@47.95.66.119
