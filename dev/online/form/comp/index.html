# 字段排序

字段排序配置：页面属性->是否排序

  • 支持多个字段，多个字段即order by datevalue, timevalue
  • 如不勾选字段，online列表查询默认以id倒序排
  • 自定义字段的排序规则

详见 ：表单控件扩展参数

点击“更多配置”->“高级配置”，直接选择排序方式即可。

# 字段顺序调整

表单和列表字段显示的先后顺序是按照表单设计顺序来展示的，直接向上或向下拖动即可

# 字段显示/隐藏控制

Online可控制字段在列表中和表单中是否显示；

# 校验规则

校验规则可扩展，支持非空，邮箱，数字、手机号正则等等校验规则。

如果选项里没有你需要的规则，你还可以通过输入正则表达式的方式来进行配置校验规则（详细配置方式可见下方的演示）。

需要注意点：如果同时校验数字，并且校验必须输入，请选择校验必填字段。

# 配置自定义校验规则

下图中配置了一段正则表达式：^123$ ，意思是只有输入了 "123" 的时候才通过校验，你也可以输入与你需求相匹配的正则表达式

“页面属性”->“更多配置”的“校验提示”中，可输入正则校验提示信息，例如：必须输入数字123
